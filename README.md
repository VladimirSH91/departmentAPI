# departmentAPI

Сервис предоставляет REST API для управления иерархией подразделений и сотрудниками. Позволяет создавать, перемещать и удалять подразделения вместе с сотрудниками, а также получать структуру с настраиваемой глубиной вложенности.

## Развёртывание в системе

python3 -m venv venv создание виртуального окружения
source venv/bin/activate активация виртуального окружения
pip install -r requirements.txt установка зависимостей

## Docker

docker-compose up -d запуск контейнеров в фоне
Кроме самого приложения в докер контейнере разворачивается БД PostgreSQL

## Как задезть вовнутрь БД в докере (подсказка для себя)

psql -h localhost -U postgres -d postgres
